Welcome to the Power BI Embedded Demo

This application demonstrates how to embed Power BI reports into a Python Flask web application using the App Owns Data (embed for your customers) pattern.

Reports

View your Power BI report embedded directly in the browser.

Open Reports
How It Works

The app authenticates via Microsoft Entra ID (Azure AD) and requests an embed token from the Power BI REST API.

Read the Docs
Quick Start
  1. Register an app in Microsoft Entra ID
  2. Gather --tenant-id, --client-id, --client-secret
  3. Get --workspace-id and --report-id from Power BI
  4. Run uv run python main.py --tenant-id=… --client-id=… --client-secret=… --workspace-id=… --report-id=…
  5. Open http://localhost:5000

Run uv run python main.py --help for all options.
