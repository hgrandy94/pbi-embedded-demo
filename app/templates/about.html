How It Works

This portal uses the App Owns Data pattern to embed Power BI reports. Your organisation's credentials authenticate with Microsoft Entra ID — end users never need a Power BI license. Here's what happens when you open a report:

  1. You sign in to this portal with your Contoso Health credentials.
  2. The portal authenticates with Microsoft Entra ID (Azure AD) using a registered service principal — your secrets never leave the server.
  3. An embed token is requested from the Power BI REST API. This token is short-lived and scoped to a single report.
  4. The token and embed URL are sent to your browser, where the Power BI JavaScript SDK renders the full interactive report in an iframe.
  5. All report interactivity — slicers, cross-filtering, drill-through, page navigation — works automatically because the iframe runs Power BI's own viewer.
